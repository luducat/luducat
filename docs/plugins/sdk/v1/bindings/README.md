# SDK Bindings

The Plugin SDK source lives at `luducat/plugins/sdk/` in the main repository.

Import via:

```python
from luducat.plugins.sdk.network import PluginHttpClient
from luducat.plugins.sdk.storage import PluginStorage
from luducat.plugins.sdk.json import loads, dumps
# etc.
```

See the [SDK Overview](../docs/overview.md) for the full module map.
