# Community Themes

Themes change the look and feel of luducat. They use standard Qt QSS
stylesheets.

## Submitting a Theme

There is no formal submission procedure yet. To list your theme here,
open an issue or pull request.

**Requirements:**
- OSI-approved open source license
- `.luducat-theme` package format (ZIP with theme.json + QSS + assets)
- Theme documentation coming soon -- see the bundled themes in
  `assets/themes/` for reference
